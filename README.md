CloudVitals | Smart Server Monitoring
CloudVitals is a lightweight web application designed to help administrators monitor their server infrastructure in real-time. This project focuses on simplifying server health tracking (CPU, RAM, Disk) through a clean, intuitive dashboard.<br>

🚀 Features
*Real-time Monitoring: Track CPU, RAM, and Disk usage instantly.

*Secure Connection: Uses SSH protocols to fetch data from remote servers securely.

*User-Friendly Interface: Modern, responsive design built for both desktop and mobile.

*Data Visualization: Dynamic charts and graphs to make sense of server performance.<br>

🛠️ Tech Stack
*Frontend: HTML5, CSS3 (Custom animations and grid layouts)

*Backend: Python (Flask Framework)

*Tools: Git, VS Code
